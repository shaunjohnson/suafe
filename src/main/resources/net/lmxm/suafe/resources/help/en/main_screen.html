Main Screen

The main screen consists of a toolbar and three tabs.

  • Toolbar
  • Users Tab
  • Groups Tab
  • Access Rules Tab
The toolbar provides quick access to general file actions as well as quick access to add users, groups and access rules. Select one of the three tabs to view and manipulate the lists of users, groups or access rules. The preview button generates an authz file and displays it for your review. Changes to the contents of the preview window are not persisted; preview is for review only.
